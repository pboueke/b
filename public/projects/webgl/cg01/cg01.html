Controls

Text:
* "Text Area": Input/Output that translates the string to the coordinates in the scene.
* "Format": [line1]; ...; [lineN]; = [(x11, y11), (x12, y12)]; ...; [(xN1, yN1), (xN2, yN2)];

Buttons:
* "Set": Uses current text to generate scene.
* "Get": Uses current scene to generate text.
* "+" : Adds random line to scene.
* "DEL key": Removes last line.

Lines:
* "Handles": Click and drag a handle to modify the selected line.
